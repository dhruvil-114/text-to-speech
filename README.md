# 🎙️ Text-to-Speech Converter

A simple and efficient Python tool that converts written text into high-quality AI voice.

## ✨ Features
- Fast and accurate text-to-speech conversion.
- Generates clear audio output.
- Simple, easy-to-use command-line/GUI interface.
